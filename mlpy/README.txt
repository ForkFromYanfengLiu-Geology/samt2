The module "mlpy_mod.py" is used as a frame for the use of SVM in SAMT2.

mply is an open source project: "http://mlpy.sourceforge.net/" which
contains some useful techniques like: Regression, Classification,
Clustering, Wavelet Submodule. A first example for usage of mlpy
is the integration of SVM training in SAMT2. Other integration is 
following in next version of SAMT2.
